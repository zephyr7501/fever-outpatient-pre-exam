# 発熱外来 予診票テンプレート

## 概要
新型コロナウイルス感染症(COVID-19)を対象とした発熱外来の予診に利用可能なテンプレートです。  
PDFのほか、Microsoft Word(.docx)のフォーマットを用意してありますので適宜修正のうえ、ご利用ください。

## 対象とする方
**発熱外来の診察を行う医療機関**および各自治体の**医師会、保健所**を対象としています。
